# PB - Forum Tecnologia 

Aplicação criada com as principais tecnologias e conceitos do .NET 6. 

Obs: API criada para ser consumida pelo projeto em MVC, separando a camada de negocio de ambas camadas. 

Tecnologias utilizadas:

-JWT AUTHETICATION;

-Banco local T-SQL;

-.NET 6;

-Entity Framework;

-Fluent Validation;

-Bloob;

-REST API;

-Visual Studio 2022;
